2.1 Installing Python

To start your Python programming journey, you need to install Python on your computer. This lesson will guide you through the installation process on various operating systems.

Windows:

  1. Visit the Python Official Website:

    • Go to Python Downloads on your web browser.

  2. Download Python Installer:

    • Click on the "Downloads" tab.

    • Choose the latest version of Python (e.g., Python 3.10.2) for Windows.

    • Select the installer that corresponds to your system architecture (usually 64-bit).

  3. Run the Installer:

    • Once downloaded, run the installer.

    • Check the box that says "Add Python X.X to PATH" during installation.

    • Click "Install Now" to start the installation.

  4. Verify Installation:

    • Open the Command Prompt and type python --version or python -V to confirm the installation.

    • You should see the installed Python version.

macOS:

  1. Check for Pre-installed Python:

    • Open Terminal and type python3 --version or python3 -V.

    • If Python is not installed, move to the next step.

  2. Install Homebrew (if not installed):

    • Follow the instructions on Homebrew to install it.

  3. Install Python using Homebrew:

    • In Terminal, type brew install python3 to install Python.

    • Verify the installation by typing python3 --version or python3 -V.

Linux:

  1. Check for Pre-installed Python:

    • Open the terminal and type python3 --version or python3 -V.

    • If Python is not installed, move to the next step.

  2. Install Python using Package Manager:

    • For Debian-based systems (e.g., Ubuntu), type sudo apt-get update followed by sudo apt-get install python3.

    • For Red Hat-based systems (e.g., Fedora), use sudo dnf install python3.

  3. Verify Installation:

    • Type python3 --version or python3 -V in the terminal to confirm the installation.

Conclusion:

Congratulations! You have successfully installed Python on your computer. In the next lesson, we'll explore writing your first Python program and understanding the basics of Python syntax. If you encounter any issues during installation, refer to the official Python documentation or seek help from the Python community.



2.2 Your First Python Program

Now that you have Python installed on your computer, let's write your first Python program. This lesson will cover the basics of creating a simple program and running it to ensure your Python setup is working correctly.

Using a Code Editor:

Choose a code editor to write and run your Python code. Here are a few popular choices:

  1. Visual Studio Code (VSCode):

    • Download and install VSCode.

  2. PyCharm:

    • Download and install PyCharm.

  3. Jupyter Notebooks:

    • If you prefer an interactive environment, Jupyter Notebooks come pre-installed with Python.

    • Open the terminal and type jupyter notebook to start a new notebook.

Writing Your First Python Program:

  1. Open Your Code Editor:

    • Launch your chosen code editor.

  2. Create a New File:

    • Create a new file with a .py extension. For example, first_program.py.

  3. Write Your Python Code:

    • In your file, type the following simple Python program:

      print("Hello, Python!")

  4. Save the File:

    • Save the file after writing your code.

Running Your Python Program:

  1. Terminal/Command Prompt:

    • Open the terminal or command prompt.

  2. Navigate to the File Location:

    • Use the cd command to navigate to the directory where your Python file is located.

      cd path/to/your/file

  3. Run the Python Program:

    • Type the following command to run your Python program:

      python first_program.py   # or python3 first_program.py on some systems

  4. Observe Output:

    • You should see the output: Hello, Python! printed on the screen.

Congratulations!

You've successfully written and run your first Python program. This simple example introduces you to the basic syntax of Python, and in the upcoming lessons, we'll explore more fundamental concepts such as variables, data types, and control flow. If you encounter any issues, double-check your code and ensure that your Python environment is set up correctly. Don't hesitate to seek help from the Python community or refer to the official documentation.



2.3 Understanding Python's Syntax

Python's syntax is designed to be clear, readable, and expressive. In this lesson, we'll explore the fundamental elements of Python syntax, including statements, indentation, comments, and basic conventions.

1. Statements and Print Function:

  • In Python, each line of code is a statement. Unlike some other programming languages, Python does not require a semicolon at the end of each statement.

    print("Hello, Python!")

    • The print() function is commonly used to display output. In the example above, it prints the text "Hello, Python!" to the console.

2. Indentation:

  • Python uses indentation to indicate code blocks. Indentation is crucial for defining the structure of your code.

    if True:
    print("This statement is indented, indicating it's part of the if block.")

    • The standard indentation is four spaces, and it's essential to be consistent throughout your code.

3. Comments:

  • Comments in Python are preceded by the # symbol. Comments are ignored during the execution of the program and are used for providing explanations or annotations.

    # This is a comment
print("This is not a comment")  # This is an inline comment

4. Variables and Assignments:

  • Variables are used to store data. Python is dynamically typed, meaning you don't need to declare the data type explicitly.

    message = "Hello, Python!"
print(message)

    • In this example, the variable message is assigned the value "Hello, Python!".

5. Basic Conventions:

  • Python follows a set of conventions to ensure code consistency and readability.

    • Variable names are typically lowercase with words separated by underscores (snake_case).

    • Function names are lowercase with words separated by underscores (function_name()).

    • Constants are written in uppercase with underscores (CONSTANT_NAME).

    • Use meaningful names to enhance code readability.

6. Case Sensitivity:

  • Python is case-sensitive. Variable names "my_variable" and "My_Variable" are different.

7. Multiple Statements on One Line:

  • While not always recommended for readability, Python allows multiple statements on one line using semicolons.

    print("Statement 1"); print("Statement 2")

Conclusion:

Understanding Python's syntax is crucial for writing clean and effective code. As we progress, we'll delve deeper into concepts such as data types, control flow, and functions. Practicing good indentation and commenting habits will make your code more readable and maintainable. In the next lesson, we'll explore variables and different data types in Python.
