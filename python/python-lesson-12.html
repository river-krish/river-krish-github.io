12.1 Introduction to Flask

Flask is a lightweight and flexible web framework for Python that simplifies the process of building web applications. In this lesson, we'll provide an introduction to Flask, covering its key features and basic usage.

1. What is Flask?

  • Definition:

    • Flask is a micro web framework written in Python. It is designed to be simple, easy to use, and scalable. Flask provides the essential tools and features needed to develop web applications without imposing a rigid structure.

  • Key Points:

    • Micro Framework: Flask is often referred to as a micro-framework because it focuses on simplicity and only includes the components necessary for building web applications. This allows developers the flexibility to choose additional tools and libraries based on their needs.

    • Extensible: While Flask provides the basics, it can be easily extended with various Flask extensions and third-party libraries to add functionalities such as authentication, database integration, and more.

    • Jinja2 Templating: Flask uses the Jinja2 template engine, allowing developers to create dynamic and reusable templates for rendering HTML content.

2. Installing Flask:

  • Use the following command to install Flask using pip:

pip install Flask

3. Hello World with Flask:

  • Create a simple Flask application with a "Hello, World!" route.

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

  • Save the code in a file named app.py and run it using:

python app.py

  • Open a web browser and navigate to http://127.0.0.1:5000/ to see the "Hello, World!" message.

4. Routes and Views:

  • Routes in Flask define the endpoints or URLs that users can visit. Views are functions associated with these routes.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return 'Home Page'

@app.route('/about')
def about():
    return 'About Page'

@app.route('/contact')
def contact():
    return 'Contact Page'

if __name__ == '__main__':
    app.run(debug=True)

5. Templates with Jinja2:

  • Use templates to render HTML dynamically.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('home.html', title='Home Page', content='Welcome to the home page.')

if __name__ == '__main__':
    app.run(debug=True)

  • Create a templates folder in the same directory as your Flask application and add an home.html file:

6. Flask Extensions:

  • Flask extensions provide additional features for your application. For example, the Flask-WTF extension can be used for handling forms.

pip install Flask-WTF

from flask import Flask, render_template
from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField

app = Flask(__name__)
app.config['SECRET_KEY'] = 'secret'

class MyForm(FlaskForm):
    name = StringField('Name')
    submit = SubmitField('Submit')

@app.route('/', methods=['GET', 'POST'])
def form_example():
    form = MyForm()
    if form.validate_on_submit():
        return f'Hello, {form.name.data}!'
    return render_template('form_example.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)

  • Create a templates folder and add a form_example.html file:

htmlCopy code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Form Example</title>
</head>
<body>
    <h1>Form Example</h1>
    <form method="POST" action="/">
        {{ form.csrf_token }}
        {{ form.hidden_tag() }}
        {{ form.name.label }} {{ form.name(size=20) }}
        {{ form.submit }}
    </form>
</body>
</html>

7. Conclusion:

Flask is a versatile and user-friendly web framework that allows developers to quickly build web applications. Its simplicity and flexibility make it an excellent choice for small to medium-sized projects. As you explore Flask, refer to the official documentation for in-depth information and best practices.





12.2 Creating a Simple Web Application with Flask

In this lesson, we'll create a simple web application using Flask. We'll cover the basic structure of a Flask application, routing, templates, and rendering dynamic content.

1. Project Structure:

Create a project folder and organize it as follows:

my_flask_app/
|-- templates/
|   |-- index.html
|-- app.py

2. Installing Flask:

Make sure Flask is installed. If not, install it using:

pip install Flask

3. Creating the Flask Application:

Create a file named app.py and add the following code:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', title='Home', content='Welcome to my Flask App!')

if __name__ == '__main__':
    app.run(debug=True)

4. Creating the HTML Template:

Inside the templates folder, create a file named index.html with the following content:

htmlCopy code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{ title }}</title>
</head>
<body>
    <h1>{{ title }}</h1>
    <p>{{ content }}</p>
</body>
</html>

5. Running the Flask App:

In the terminal, navigate to the project folder (my_flask_app) and run the Flask app:

python app.py

Visit http://127.0.0.1:5000/ in your web browser to see the home page of your Flask app.

6. Adding Additional Routes:

Expand the app.py file to include more routes:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', title='Home', content='Welcome to my Flask App!')

@app.route('/about')
def about():
    return render_template('index.html', title='About', content='Learn more about my Flask App.')

@app.route('/contact')
def contact():
    return render_template('index.html', title='Contact', content='Contact us for more information.')

if __name__ == '__main__':
    app.run(debug=True)

7. Enhancing Templates:

Update the index.html template to include navigation links:

htmlCopy code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{ title }}</title>
</head>
<body>
    <nav>
        <ul>
            <li><a href="{{ url_for('home') }}">Home</a></li>
            <li><a href="{{ url_for('about') }}">About</a></li>
            <li><a href="{{ url_for('contact') }}">Contact</a></li>
        </ul>
    </nav>
    <h1>{{ title }}</h1>
    <p>{{ content }}</p>
</body>
</html>

8. Conclusion:

Congratulations! You've created a simple web application using Flask. This basic structure can be expanded upon to build more complex applications. Explore Flask's documentation for additional features, such as handling forms, working with databases, and creating dynamic routes. As you become more familiar with Flask, you'll have the flexibility to develop web applications tailored to your specific needs.
